elenes dotfiles
===============

Setup
-----

	git clone https://github.com/elenes/dotfiles.git

### Linux / Mac OS

To install vim files properly:

	cd dotfiles
	sh install.sh

### Windows

	cd dotfiles
	git init
	git submodule init
	git submodule update
	
Copy `.vim/pathogen/autoload/`
to   `.vim/autoload/`

Rename/move stuff as needed
